# 博客Chrome拓展插件

## 使用说明

1. 克隆或下载项目到本地

![avatar](assets/step1.png)

2. Chrome浏览器打开拓展程序，点击加载已解压的拓展程序

![avatar](assets/step2.png)

3. 选择项目的 `src` 文件夹即可使用
